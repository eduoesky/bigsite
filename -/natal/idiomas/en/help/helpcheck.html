What are the verification reasons?
The verification reasons are the reasons for which you are reporting a Pou. These are: Fake likes, Pou robotized y Disturbance, but what is it?

Fake likes: Fake likes It's when a Pou creates accounts (manually or automatically) to give likes to themselves. If you see an account of this kind, check this option. It can be harmful when the account exceeds 30 fake likes.
Like robotized: Like robotized It's when a Pou has robotized likes, meaning likes that are not genuine and are entirely created from scratch, like robots. If you think a Pou is like this, check this option.
Pou robotized: un Pou robotized It's when a Pou has their account made up of robots, where they automatically click on messages (guestbook), visit, like other Pous/remove the like, send many messages, change clothes repeatedly... If you think a Pou is like this, check this option.
Disturbance: Disturbance "It's when a user (not necessarily a Pou) is sending offensive messages, insulting, or any such cases (not valid for normal conversations). If you think someone is like this, check the option and write the Pou name of the user. If there are 2, write both, separating them only by commas.
