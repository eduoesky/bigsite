Welcome to the Pou name marker! ^_^

Read the rules:

To mark the name of your Pou on the board, you must follow the following rules:
1. No profanity
It is crucial to maintain respect in words, so do not use profanity.
2. Do not harm!
If the Pou's name you are marking is not your own, do not mark it on the board; you may harm the player when editing or if they do not want the nickname on the board.
3. Mention the name correctly
If your name has any lowercase letters or any observations (e.g., my Pou's name is written with 'O' and 'K' in uppercase in 'OesKy').
4. Minimum and maximum limit of sentences
For each sentence, the required minimum is at least 20 characters and a maximum of 130.
5. Avoid abbreviations
If you use an abbreviation that is not recognized, the sentence will not be added.
6. Do not include the Pou's name in the password
When entering the access password for the Pou's name, never include the Pou's password, or their name together, or any of your data.
7. Be respectful
Maintaining respect for others is always good! xD - if someone disrespects your Pou, please let me know here > CHECK PAGE

OPTIONAL :)
1. Emojis
It is possible to use emojis :), literally any, unless it is disrespectful. Disrespectful emojis will be replaced by others. Maximum limit of emojis: 9
2. Name of another Pou
If you have 2 or more Pous, feel free to put the names of your Pous together with yours. REMEMBER: The sentence CANNOT have only the names of the Pous; type something before or after.
3. Choose between password and code
A 15-character code is generated for you to save the Pou's name. The password is something you create with personal hints that you remember, so make the right choice; if you forgot your code or password, click 'Forgot my password or code' on the editing page, and enter the email you provided when marking your Pou's name. However, the choice between the code and the password is entirely optional, but you cannot enter both at the same time!!!

IF I DIDN'T PUT MY POU'S NAME, BUT IT APPEARS ON THE BOARD, WHAT TO DO??
In these cases, you have to show that the Pou is yours, and then I can take necessary action :)

NOTES
1. If your Pou's name has already been marked on the board, only that time will be allowed. That is, marking the name of the same Pou twice will not result in anything, not even in changes of positions, unless it is edited.
2. This service works only to register Pou names, so whether I like it or not, it will not prevent your Pou's name from being marked!
3. Names of Pous that have been banned from this site will not be marked
4. As I mentioned before, uppercase and lowercase matter, meaning you can edit the name of a Pou that has a wrong uppercase/lowercase!
